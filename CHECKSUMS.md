# SHA256 Checksums

This file contains SHA256 checksums for all APK files in this repository. Use these to verify the integrity of downloaded files.

## How to Verify

### On Linux/macOS/Termux:
```bash
sha256sum com.termux_*.apk
# Compare output with checksums below
```

### On Windows (PowerShell):
```powershell
Get-FileHash -Algorithm SHA256 com.termux_*.apk
# Compare output with checksums below
```

## Why Verify Checksums?

Verifying checksums ensures:
- The file downloaded completely without corruption
- The file hasn't been tampered with
- You have the exact same file as hosted in this repository

Always verify checksums when downloading APK files from any source for security.
